# Smart Glasses for Visually Impaired

## Description
This project is designed to help visually impaired people by detecting obstacles and providing alerts using smart glasses technology.

## Features
- Object detection
- Voice alert system
- Real-time assistance

## Technologies Used
- Python
- Machine Learning
- OpenCV

## How to Run
1. Install required libraries
2. Run the main Python file
